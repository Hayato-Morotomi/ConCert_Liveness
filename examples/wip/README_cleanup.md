# Rocq 日本語コメント削除ツール

Rocq (`.v`) ファイルから日本語を含むコメントのみを削除します。

## 主な機能
- **選択的削除**: 日本語を含むコメントのみを削除し、英語コメントは維持します。
- **解析精度**: 文字列内のエスケープされた引用符 (`""`) や、コメントの入れ子構造を正しく判別します。
- **行番号維持**: 削除箇所を空行に置き換えるため、コンパイルエラー時の行番号が狂いません。

## 使い方
1. **シミュレーション**
   `python3 clean_jp.py --dry-run`
2. **実行 (バックアップ作成)**
   `python3 clean_jp.py`
3. **バックアップの削除 (動作確認後)**
   `find . -name "*.v.bak" -delete`

## 注意事項
パースエラー（閉じられていないコメント等）があるファイルは安全のためスキップされます。

# 実行権限の付与
chmod +x clean_jp.py

# カレントディレクトリ以下の全ファイルをスキャンして実行
python3 clean_jp.py

# 特定のディレクトリ配下のみ実行
python3 clean_jp.py ./theories/Internal

# .bak ファイルをすべて削除する
find . -name "*.v.bak" -type f -delete

# もし一括で元に戻したい場合 (バックアップがある場合)
find . -name "*.v.bak" | sed 's/\.bak//' | xargs -I {} mv {}.bak {}
